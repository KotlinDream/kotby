## 背景

在编写Kotlin项目的时候，感觉一些基本类型的方法还是不如Ruby方便，所以想在kotlin中实现Ruby的一些便捷方法

## 使用

```kotlin
"aBc".camelize() // "Abc"
"abc_edf".camelize() // "AbcEdf"
```
